READ ME
-------------

Prerequisites:
------------------
1. Visual Studio 
	User should have Visual Studio installed on his/her machine.
	Current version is created in Visual Studio 2012.

2. SQL Server
	User should have SQL server installed on his/her machine.

------------------------------------------------------------------------------------------------------------------
Technology Stack:
----------------------
Project is created using C#, MVC 4, SQL Database, WEB API, JQUERY and Entity Framework.
------------------------------------------------------------------------------------------------------------------

Steps To Run:
----------------------
Open Project in Visual Studio
Resolve nuget dependencies if any.

Please change host address in web.config file.
Go to AppSettings section and change value of WebAPICallURL, currently it is set to http://localhost:55174.
Set it according to your machine and Run the project.
